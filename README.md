### **Installation Steps**
## Install Addon

1. Log in to your **WBCE CMS Admin Panel**.
2. Go to **Add-ons > Modules**.
3. Click **Upload**.
4. Select the **All in One Accessibility** add-on ZIP file.
5. Click **Upload** and wait for the installation to complete.

## Configuration

1. After the installation is complete, go to **Admin Tools**.
2. Click **All in One Accessibility**.
3. Configure the widget settings as required.
4. Save the settings.
